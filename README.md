Slide 2: Introduction 
Project Overview 
A Health Data Analytics System is a software solution used to collect, manage, process, and 
analyze patient and hospital information efficiently using Object-Oriented Programming (OOP). 
In real hospitals and clinics, large amounts of patient data are handled daily, including patient 
registration, doctor assignments, appointment scheduling, diagnosis tracking, and health 
monitoring. Managing this manually can lead to errors, delays, and poor decision-making. 
This project provides a simple prototype that demonstrates how programming can be used to 
automate these tasks using Python. 
Main Goal 
To create a working system that demonstrates: 
• Class creation and object design 
• Encapsulation and abstraction 
• Inheritance and polymorphism 
• Object interaction and communication 
• Conditional logic and workflow control 
• Health data analysis and reporting 
This project satisfies the requirements for Milestone 1 and Milestone 2.

Explanation 
After building the basic classes in Milestone 1, the next step is to make those objects interact like 
they do in real life. 
For example: 
• A patient books an appointment with a doctor 
• A doctor reviews the patient’s condition 
• The system stores the patient’s health record 
• The health record analyzes sugar level and blood pressure 
• The system gives a health status result automatically 
This stage makes the system dynamic and functional instead of just static class definitions. 
Objective 
Create a functional multi-object interacting system.
